Privacy Policy

Privacy Policy

Effective Date: December 13, 2025

1. Our Commitment to Privacy

The EXIF Image Date and Time Changer is a fully client-side web tool that allows you to correct or add EXIF capture dates to JPEG images. We do not collect, store, transmit, or process any personal data or images. All operations occur entirely within your browser, ensuring your photos and metadata remain private and never leave your device.

2. No Data Collection

We intentionally designed this tool to operate without any data collection. Specifically, we do not gather:

  • Personal information such as names, email addresses, or IP addresses
  • Image content or any uploaded files
  • EXIF metadata from your photos
  • Usage statistics, analytics, or tracking data
  • Cookies, local storage persistence, or session identifiers

Because everything processes locally using JavaScript, your data never touches any server.

3. How the Tool Functions

When you use the EXIF Image Date and Time Changer:

  • Images are read directly in your browser using the File API
  • EXIF data is parsed and modified using client-side libraries
  • Corrected images are generated as downloadable files on your device
  • Batch ZIP creation happens locally using browser capabilities

No information is ever sent externally. Temporary memory usage clears when you close the tab or refresh the page.

4. Third-Party Services

We do not integrate any third-party analytics, advertising, or tracking services. The tool uses only open-source JavaScript libraries loaded via CDN for functionality, but these do not track or collect user data.

5. Security and Responsibility

Since no data is transmitted, there is no server-side risk. We recommend using the tool on trusted devices with up-to-date browsers for optimal security. You remain responsible for the security of your local environment and any downloaded files.

6. Children's Privacy

This tool is not directed at children under 13. We do not knowingly collect any information from children.

7. Changes to This Policy

We may update this Privacy Policy occasionally. Changes will be reflected on this page with an updated effective date.

8. Contact

For questions about this policy, please open an issue on our public GitHub repository.

Your privacy is our priority — we built this tool to respect it completely by collecting absolutely nothing.
